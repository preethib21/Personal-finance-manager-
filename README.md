# Personal Finance Manager

A Python-based Personal Finance Manager that helps users track income, expenses, and savings efficiently. The application allows users to record financial transactions, view spending details, and monitor their overall financial status.

## Features

- Add income records
- Add expense records
- View transaction history
- Calculate total income and expenses
- Display current balance
- Store financial data using file handling

## Technologies Used

- Python
- File Handling
- Functions
- Lists
- Conditional Statements
- Loops

## How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Open the project folder.
4. Run the Python file:

```bash
python finance_manager.py
```

## Project Outcome

This project demonstrates practical knowledge of Python programming, file management, data organization, and financial record tracking through a menu-driven application.
